k-frame has been designed from the start for gradual adoption, and you can use as little or as much k-frame as you need. Whether you want to get a taste of k-frame, add some interactivity to a simple HTML page, or start a complex k-frame powered app, the links in this section will help you get started.

Create a New k-frame App

When starting a k-frame project, a simple HTML page with script tags might still be the best option. It only takes a minute to set up!

As your application grows, you might want to consider a more integrated setup. There are several JavaScript toolchains we recommend for larger applications. Each of them can work with little to no configuration and lets you take full advantage of the rich React ecosystem.
